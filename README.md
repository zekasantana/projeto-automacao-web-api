# Projeto de Automação

## Tecnologias

- Cypress
- Cucumber
- JavaScript

## Instalação

```bash
npm install
```

## Executar testes

```bash
npx cypress open
```

ou

```bash
npx cypress run
```

## Cenários automatizados

- Login
- Busca de produto
- Adicionar produto ao carrinho
- Validar carrinho
- API Trello
